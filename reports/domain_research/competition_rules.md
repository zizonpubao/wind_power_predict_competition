## 제출 / 팀 규정 (공식 `/overview/rules` 페이지 원문 기준)

- **1일 최대 제출 횟수**: 5회
- **팀 최대 인원**: 3명
- **중복 참가 금지**: 개인 참가자가 동시에 팀원으로 등록하거나, 한 사람이 여러 팀에 동시에
  속할 수 없음.
- **팀 병합**: 2026-08-07(금) 23:59까지 가능 (위 일정표 참고). 병합 후 인원 제한(3명) 초과
  여부 등 세부 조건은 규정 페이지에 별도 항목이 있을 수 있으나 이번 조사에서 추가 확인은
  못함 — 필요시 원문 재확인 권장.

## 금지 데이터 / 금지 방법

- **평가 구간 정답 관련 정보 금지**: "평가 구간의 실제 발전량, 실측 운영 데이터, 비공개 운영
  데이터 또는 정답에 준하는 정보는 사용할 수 없습니다." — `docs/data_description.md` 12번
  항목("평가 기간의 실제 발전량, SCADA, 비공개 운영 데이터, 사후 보정자료, 재분석자료 등은
  예측에 사용할 수 없습니다")과 사실상 동일한 취지이며, 공식 규정 페이지가 이를 재확인한다.
- **사후 관측/재분석 자료 금지**: 예측 시점 이후에 생성된 관측 데이터나 재분석(reanalysis)
  산출물 사용 금지. (data_description.md의 "재분석자료" 언급과 일치)
- **외부 API/원격 추론 모델 금지**: "OpenAI API, Gemini API, Hugging Face Inference API" 등
  원격 추론 서비스 사용 금지. **로컬에서 직접 실행되는 모델만 허용.** 이는
  `data_description.md`에는 없는, 공식 규정 페이지에서 새로 확인된 추가 제약이다.
- **모델 라이선스/공개 시점 제약**: 2026-07-06(대회 시작일) 이전에 가중치가 공개된
  오픈소스 모델만 사용 가능하며, 상업적 이용이 허용되는 라이선스여야 함. 이 역시
  `data_description.md`에는 없던 규정 페이지 고유 규정.

  명시는 원문에서 추가 확인 필요).

## 2차 발표평가 (2차 평가) 관련

리더보드 상위팀을 대상으로 한 2단계 평가 구조:

1. **1차 (온라인 리더보드)**: `sample_submission.csv` 형식의 예측 CSV를 대회 기간
   (~08-14 10:00) 중 제출. 리더보드 점수는 **Private Score(테스트 데이터의 60%)** 기준으로
   산출된다고 확인됨(평가 페이지 요약).
2. **2차 대상자 선정**: 리더보드 상위 **30팀(예비 10팀 포함)** 이 2차 평가 대상.
   2026-08-14 12:00 ~ 08-17 10:00 사이에 코드, 모델, 산출물을 제출해야 함.
3. **산출물 검증**: 2026-08-17~08-21 사이 제출된 코드/모델의 재현성 등을 검증. 2차 평가
   대상팀은 다음을 제출해야 하는 것으로 확인됨: 사용한 모든 외부 데이터 파일 및 출처,
   수집 방법/시점, 전처리 코드, 모델 사양, 라이선스, 그리고 제출한 자료만으로 결과를
   재현할 수 있다는 증빙.
4. **검증 통과 후 발표평가 대상**: 검증을 통과한 **상위 20팀**이 2026-08-28(금) 오프라인
   발표 평가에 참여 (과제이해도/기술우수성/문제해결력/적용가능성 평가, CLAUDE.md에도 기재됨).
5. **최종 순위 산정**: "리더보드 성과(50%) + 발표 평가(50%)"를 결합해 최종 수상 10팀을
   결정한다고 확인됨(평가 페이지 요약 — 원문 표현이 "리더보드 성과"라는 용어를 그대로 썼는지,
   아니면 1차 온라인 점수(1-NMAE/FICR 총점)를 지칭하는 것인지는 재확인 필요. 아래
   "확인되지 않은 것" 참고).

## 확인되지 않은 것 (명시)

- `/overview/prize` (상금 배분 상세), `/overview/agreement` (동의사항) 페이지는 이번 조사에서
  fetch하지 않았다 — 필요시 추가 조사 가능.
- 팀 병합 시 인원 제한 예외 여부, 실격 판정의 구체적 기준(예: 소급 적용 범위, 이의제기 절차)은
  규정 페이지 요약에서 세부적으로 확인되지 않았다.
- "리더보드 성과 50% + 발표평가 50%"에서 "리더보드 성과"가 정확히 1차 온라인 총점
  (`0.5×(1-NMAE) + 0.5×FICR`, `nmae_formula.md` 참고)과 동일한 것인지, 별도 재계산되는지는
  명시적으로 재확인하지 못했다(요약 답변에 근거, 원문 전체 재확인 권장).
- Private Score가 "테스트 데이터의 60%"라는 것은 확인되었으나, 나머지 40%(Public Score로
  추정)의 용도(중간 순위 공개용인지 등)는 이번 조사에서 세부 확인하지 않았다.
